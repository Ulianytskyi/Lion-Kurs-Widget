# Lion Kurs Desktop Widget

Компактний віджет для робочого столу Windows, який відображає актуальний курс валют з сайту [lion-kurs.rv.ua](https://lion-kurs.rv.ua/).

## Особливості
* **Завжди зверху**: Віджет закріплюється поверх усіх вікон (зручно розмістити на панелі завдань).
* **Актуальні дані**: Автоматичне оновлення курсу щогодини.
* **Інтерактивність**: При наведенні курсору відображається додатковий курс EUR та PLN.
* **Гнучкість**: Вибір основної валюти (USD, EUR, PLN) через контекстне меню (права кнопка миші).
* **Персоналізація**: Збереження позиції вікна та вибраних налаштувань після перезавантаження.

## Технології
* **Python 3.x**
* **Tkinter** — для графічного інтерфейсу.
* **BeautifulSoup4** — для парсингу даних.
* **PyInstaller** — для створення .exe файлу.

## Як запустити
1. Склонуйте репозиторій.
2. Встановіть залежності: `pip install -r requirements.txt`.
3. Запустіть скрипт: `python kurs.py`.

Або скористайтеся готовим `.exe` файлом у розділі **Releases**.
