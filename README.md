# 🦁 Lion Kurs Desktop Widget v1.2.0

Професійний десктопний віджет для моніторингу курсу валют у м. Рівне з інтегрованою аналітикою. Дані парсяться в реальному часі з сайту [lion-kurs.rv.ua](https://lion-kurs.rv.ua/).

## 🌟 Що нового у версії 1.2.0
* **📈 Візуалізація даних**: Побудова графіків зміни курсу прямо з контекстного меню (використовується `matplotlib`).
* **📜 Система логування**: Автоматичне збереження історії всіх змін курсу у файл `kurs_history.log`.
* **🔔 Розумні сповіщення**: Звуковий сигнал (Beep) при різких стрибках курсу (понад 10 копійок).
* **🎨 Динамічний дизайн**: Текст змінює колір на зелений (ріст) або червоний (падіння) залежно від тренду.

## ✨ Основні можливості
* **Завжди зверху (Topmost)**: Віджет закріплений поверх усіх вікон.
* **Smart Popup**: Наведення миші показує курси інших валют (EUR, PLN) у компактному вікні.
* **Кастомізація**: Вибір основної валюти (USD/EUR/PLN) та збереження позиції вікна після перезапуску.
* **Легкість**: Оптимізована робота у фоновому потоці (`threading`), що не навантажує систему.

## 🛠 Технологічний стек
* **Python 3.10+**
* **Tkinter** — GUI інтерфейс.
* **BeautifulSoup4 & Requests** — надійний парсинг даних.
* **Matplotlib** — побудова графіків.
* **Logging** — збереження історії операцій.

## Журнал змін (Changelog)

[1.2.0] — 2026-03-30
Додано:

Графік динаміки курсу: Можливість перегляду історії змін обраної валюти прямо з контекстного меню (реалізовано через matplotlib).

Система логування: Автоматичне збереження кожної зміни курсу у файл kurs_history.log.

Звукові сповіщення: Додано аудіосигнал (Beep) при зміні курсу понад 0.10 грн.

Багатопотоковість: Побудова графіка винесена в окремий потік (threading), щоб віджет залишався чуйним.

Виправлено:

Виправлено скидання кольору тексту при зміні активної валюти.
